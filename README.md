<br/>

A multithreaded Deep Packet Inspection backend engine in C++ that processes real-time network traffic, performing packet parsing, classification, and rule-based filtering at scale.

| Attribute | Detail |
|---|---|
| **Stack** | C++, Multithreading, Networking |
| **Scale** | Real-time processing of live network traffic across multiple concurrent flows |
| **Performance** | Worker thread pool architecture improving throughput and CPU core load balancing |
| **Security** | TLS SNI extraction & 5-tuple flow tracking for app identification without payload decryption |
| **Impact** | Enables encrypted-traffic application identification without compromising payload privacy |
| **Repository** | [github.com/Mansi176-dotcom/packet-analyzer](https://github.com/Mansi176-dotcom/packet-analyzer) |

Architected a concurrent processing pipeline using worker thread pools and lock-protected, thread-safe queues, enabling efficient classification of network flows at scale while balancing load across CPU cores.

</details>

| Attribute | Detail |

| **Security** | REST API layer designed for safe, controlled access to indexing & retrieval functionality |
| **Impact** | Powers a fully local RAG pipeline for document Q&A without external API dependencies |
| **Repository** | [github.com/Mansi176-dotcom/own-ai-project](https://github.com/Mansi176-dotcom/own-ai-project) |

Designed and exposed indexing and retrieval functionality through REST APIs, enabling integration with downstream applications, and built tooling to benchmark HNSW, KD-Tree, and brute-force strategies for accuracy and speed trade-offs.

</details>
